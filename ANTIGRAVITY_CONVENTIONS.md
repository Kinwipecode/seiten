# Antigravity Projekt-Konventionen (Landingpage)

Dieses Dokument definiert spezifische Befehle und Arbeitsweisen für den KI-Assistenten in diesem Projekt-Ordner (`01_Meine Seiten Online`).

## 🤖 Benutzerdefinierte Befehle (Custom Commands)

Wenn der Benutzer einen der folgenden Befehle gibt, führe automatisch die verknüpften Aktionen aus:

- **Befehl: "Update" oder "Aktualisieren"**
  - **Aktion:** Führe exakt die Schritte aus, die in der Datei `UPDATE_ANLEITUNG.md` im Hauptverzeichnis beschrieben sind.
  - **Ziel:** Lade die neuesten Repositories über die GitHub API, aktualisiere die `index.html` und pushe die Änderungen ohne weitere Rückfragen live.
